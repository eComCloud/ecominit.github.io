NAME

eci.auxinit —
auxiliary process control initialisation

SYNOPSIS

eci.auxinit

DESCRIPTION

eci.auxinit is the equivalent of eci.init(8) for auxiliary eComInit instances. It carries out the same tasks except those specific to system initialisation. For example, it does not read /etc/ttys nor /etc/inittab.
