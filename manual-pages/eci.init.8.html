NAME

eci.init —
process control initialisation

SYNOPSIS

eci.init

eci.init [0 | 1 | 6]

DESCRIPTION

eci.init is the first process run on a system managed by eComInit. Its exact tasks vary by platform, but it is generally reserved to setting up the system just enough to launch and start eci.startd(8), restarting it if it should fail. To preserve backwards compatibility, it also supervises processes specified in either the /etc/ttys file on BSD-like systems, or the /etc/inittab file on System V-like systems.
